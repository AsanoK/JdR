Structure et chronologie
D-21 : libération des TITANS dans le cyberconflit USA-RPC
D-14 : contamination exsurgente des TITANs
D-7 : premiers projets illégaux menés par les TITANs
D-5 : crise humanitaire en Inde, risque majeur de famine suite à un problème de programmation de nanofabricateurs; la société Imprint refuse tout commentaire
D-4 : Une base militaire Russe en Prusse est attaquée par un groupe terroriste inconnu, le Kremlin voit la fédération nord-africaine derrière ces actions, afin de destabiliser le jeu des pouvoirs en Europe.
D-3 : le cyberconflit USA-RPC risque de s'embraser quand le porte-aéronef L-2 est touché par un tir inconnu dans le port de Zhanjiang. La RPC annonce l'implication d'un sous-marin américain. Le Pentagone dément toute implication. La Russie annonce avoir abattu un aéronef inconnu dans son espace aérien. Les réseaux commencent à avoir quelques problèmes mineurs
D : De nouvelles émeutes dans les quartiers LowSecurity au Kenya.L'armée est envoyée pour calmer les émeutiers. Les émeutes auraient pour cause des rumeurs de disparitions dans les LS. Etat d'urgence dans plusieurs pays.
D+1: frappes de décapitations visant le Kremlin, les USA et la RPC, entre autres. Premières offensives TITANs ouvertes, les armées nationales se battent contre elles-mêmes. Les réseaux tombent rapidement
D+2 : premières images de créatures exsurgentes, début des opérations contre les grandes villes. Les gouvernements n'ont plus d'influence ou presque. Certains pays annoncent une évacuation de leurs habitants sous forme de sauvegarde. Retour de quelques réseaux.
D+3 : les combats continuent dans les grandes villes, face à de nouvelles horreurs, les évacuations massives commencent là où cela est possible. l'ONU annonce une évacuation générale
D+4 : la résistance continue, Chicago est rasé par une bombe à antimatière plantée par des forces spéciales américaines.
D+5 : chute de l'ascensceur orbital après plusieurs jours d'âpres combats
